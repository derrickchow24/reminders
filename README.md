# Derrick's Reminder System

Texts +1-650-296-2919 at 8:00 AM PST for any date you add.

## Add a reminder
```
node index.js add "9/3 - Mom's birthday"
node index.js add "12/25 - Christmas"
```

## See all reminders
```
node index.js list
```

## Send a test text
```
node index.js test
```

## Run the scheduler (do this on Railway)
```
node index.js
```
